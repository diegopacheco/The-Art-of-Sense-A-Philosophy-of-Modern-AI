# Prompt Library

Claude Code has a prompt library. When you have a powerfull LLM sometimes the GAP it's not use the LLM but knowing how you can use the LLM, meaning what things the LLM can do, so you need to learn how to know which questions to ask, sometimes theese questions are called "prompts".

[Claude Prompt Library](https://docs.claude.com/en/resources/prompt-library/library) it's a list of questions or prompts if you will, that allow you to see things claude can do.
