This is an mdbook project for "The Art of Sense: A Philosophy of Modern AI" by Diego Pacheco.

## Read the Book in Git

[Read it Here](src/README.md)

## Prerequisites

Install mdbook:
```bash
cargo install mdbook
```

## Running

To serve the book locally:
```bash
./run.sh
```

This will start a local server and open the book in your browser.

## Structure

The book is organized into the following sections:

- Zero
- Chapters 1-9
- Epilogue

All markdown content is located in the `src/` directory with subfolders for each section.
