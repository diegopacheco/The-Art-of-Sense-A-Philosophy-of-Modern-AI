# Claude Bash Orchestration

claude can be run as a unix/linux process in bash. Meanking you can do pipelines, orchestration and all sort of bash scripting with claude code. In order to run claude in bash you just do:

```bash
claude -p "Your prompt here"
```

This is simple but extreme powerful. Because you can create complex orchestration with claude code in bash scripts. You can chain multiple claude calls, you can use the output of one claude call as the input of another claude call. You can do loops, conditionals, and all sort of bash scripting with claude code.
