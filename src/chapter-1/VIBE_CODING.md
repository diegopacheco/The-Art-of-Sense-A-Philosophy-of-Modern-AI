# Vibe Coding

Andrej Karpathy coined the tern "vibe coding" in one of his [tweets](https://x.com/karpathy/status/1886192184808149383?lang=en) on FEB 2025.

<img src="../images/01-Vibe-Coding-2025-term.png" alt="Vibe Coding Tweet" style="max-width: 600px;"/>

Vibe coding is a pratice where you generate prompts and you `do not look at the code`. You assume AI will take care of everything. Vibe code is very like try-and-error, you can build games with it even but dont foll yourself you cant use vibe coding for all the things.
