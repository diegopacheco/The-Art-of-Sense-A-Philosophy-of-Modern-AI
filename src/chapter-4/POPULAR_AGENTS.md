# Popular Agents 

There any many coding agents out there. I try and [POC most of them](https://github.com/diegopacheco/ai-playground/tree/main/agents). Here is a list of the most common and useful coding agents. I recommen you play with this coding agents, them you can see which one ones you like the most.

## Github Copilot

[Github Copilot](https://github.com/features/copilot) One of the first, if not the first coding agent. It is embeded into VSCode and other IDEs. It has a great developer experience. Copilot is on constant evolution. Copilot has one nice thing that is support for many LLM models.

## Codex CLI

There are two flavors of Codex. There is the [web sandbox version](https://chatgpt.com/features/codex) and there is the [CLI version](https://github.com/openai/codex). Codex is a good agent where they have they own LLM Model which is different from Chat GPT model.

## OpenCode

[Opencode](https://github.com/sst/opencode) it's a open souce coding agent that works with many LLM models. 

## Jules

[Jules](https://jules.google/) it's a web sandbox agent by Google backed by Gemini LLM models.

## Kiro

[Kiro](https://kiro.dev/) it's a coding agent by AWS. Kiro it's a fork of VSCode and has a great developer experience. Kiro is different because implement [Spec Driven Development (SDD)](https://martinfowler.com/articles/exploring-gen-ai/sdd-3-tools.html).

## Claude Code

[Claude Code](https://www.claude.com/product/claude-code) by anthropic is another coding agent. IMHO it's the best agent coding tool out there. Claude code works with anthropic LLM models.
